## 特徴

### **Sea Mode (海モード)**
* ランダム配置の常識とか、あたしがぶち壊すし？
* 「海」「砂浜」「陸」で構成された群島生成アルゴリズムが盤面をめちゃくちゃ生意気に彩っちゃうの♡  
* 地形読みっていう新しい戦略…理解できるかなぁ？できないなら教えてあげてもいいけど♡♡

### **Logic Bot (論理演算ボット)**
* 変なブラックボックスAIみたいに誤魔化さないもん
* 数学的ロジックで盤面をスキャンして、美しく解析して、迷いなく開けちゃう♡  
* 「島攻略（角優先）」とか戦略プロファイルも切り替えられるから、あんたが悩んでる間に全部片付けちゃうし？

### **UI/UX**
* GTAっぽい "WASTED" / "MISSION PASSED" 演出で、クリアも死亡も華やかに飾ってあげる♡  
* Modern / Sea / Classic の3テーマで、気分に合わせて雰囲気変えて遊べちゃうんだよ？
* 盤面サイズ、爆弾密度、演算速度…全部あたしの手のひらの上
